# pomoduo-privacy

Privacy policy for the [PomoDuo](https://github.com/arieltyson/PomoDuo) iOS app, hosted via GitHub Pages for App Store Connect.

**Live URL:** [arieltyson.github.io/pomoduo-privacy](https://arieltyson.github.io/pomoduo-privacy/)

## Overview

PomoDuo collects the absolute minimum data needed to function. This page documents what we collect, what we don't, and how data is handled:

- **Anonymous authentication** — no email, name, or password required
- **Ephemeral session data** — deleted from Firestore when each session ends
- **Local-only history** — session history is stored on-device via SwiftData and never uploaded
- **Opaque app tokens** — Screen Time selections are managed by iOS; we cannot see which apps are blocked
- **No analytics, no ads, no third-party tracking**
